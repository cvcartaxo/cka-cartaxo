# CKA Practice 

Straightforward CKA practice labs derived from the CKA-PREP playlist. Every question lives in its own folder with three bash files:

- `LabSetUp.bash` - copy/paste into Killercoda https://killercoda.com/ (or any Kubernetes cluster) to prep the environment.
- `Questions.bash` - the scenario text plus the YouTube link for the walkthrough.
- `SolutionNotes.bash` - a step-by-step solution when you need a hint.

## How to Use
1. Launch the CKA Killercoda playground or your own cluster.
2. Clone this repo inside the environment.
3. Pick a folder under `Question-*`.
4. Run `./scripts/run-question.sh Question-01` or cd ~/CKA-PREP-2025-v2
bash scripts/run-question.sh "Question-9 Network-Policy" to apply the setup and print the question text, or run `bash Question-01/LabSetUp.bash` manually.
5. Work through the task, then consult `SolutionNotes.bash` if you need help.

# Prática CKA 

Laboratórios de prática CKA diretos, derivados da playlist CKA-PREP. Cada questão reside em sua própria pasta com três arquivos bash:

- `LabSetUp.bash` – copie/cole no Killercoda https://killercoda.com/ (ou em qualquer cluster Kubernetes) para preparar o ambiente.
- `Questions.bash` – o texto do cenário mais o link do YouTube para o passo a passo.
- `SolutionNotes.bash` – uma solução passo a passo para quando precisar de uma dica.

## Como Usar
1. Inicie o playground CKA Killercoda ou seu próprio cluster.
2. Clone este repositório dentro do ambiente.
3. Escolha uma pasta em `Question-*`.
4. Execute `./scripts/run-question.sh Question-01` ou `cd ~/CKA-PREP-2025-v2`
`bash scripts/run-question.sh "Question-9 Network-Policy"` para aplicar a configuração e imprimir o texto da questão, ou execute `bash Question-01/LabSetUp.bash` manualmente.
5. Trabalhe na tarefa e, se precisar de ajuda, consulte `SolutionNotes.bash`.
